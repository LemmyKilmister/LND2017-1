Iconos

¿Qué son los font-awesome?

El concepto que tengo yo sobre los font-awesome tiene 3 posibles contextos.

El primer contexto son los llamados archivos SVG los cuales no son otra cosa que una imagen vectorial la cual está definida bajo un lenguaje de marca estandarizado por W3C.

El segundo contexto que tengo sobre los font-awesome es el uso directo de la etiqueta <svg> la cual está igualmente definida como en el caso anterior, con la diferencia que el código del mismo está implícito en el mismo código fuente de la web

El tercer y último contexto que le doy es el uso de la galería de código abierto Font-awesome Icon.Con este un pequeño framework podemos implementar rápidamente en nuestro website una gran variedad de iconos los cuales en los 3 contextos son más rápidos de visualizar y los mismo pueden ser reescalados sin pérdida de calidad gracias a que son imágenes de tipo vectorial.

Fuentes externas

¿Cómo incluir los tipos de letra de Google-Fonts?

Para poder incluir una nueva fuente de Google-Fonts en nuestro sitio web existen dos métodos muy sencillos.

Para incluirla por el modo estándar deberemos de usar la etiqueta link en el head del HTML como se muestra en la sección del código inferior.


	<link href="https://fonts.googleapis.com/css?family=Spectral+SC" rel="stylesheet">

Para hacerlo por el metodo @import deberemos de incluir la linea que vemos en el codigo infetrior entre la etiqueta <style> o tambien dentro del mismo archivo de estilo css de nuestro web site.


	@import url('https://fonts.googleapis.com/css?family=Spectral+SC');

Una vez hayamos realizado uno de los dos métodos para invocar la nueva fuente tan solo deberemos de llamar a la fuente como podemos ver en el condigo inferior. Recuerda que este último código se debe incluir dentro de las etiquetas <link> o dentro del mismo archivo css como en el caso anterior


	font-family: 'Spectral SC', serif;
