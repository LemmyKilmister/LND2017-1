Resets

¿Qué es un reset en CSS?

Bajo mi punto de vista un reset para CSS son una serie de parámetros que incluimos en la hoja de estilo de nuestro web site para evitar esas incomodas reglas que pone cada explorador, a la hora de diseñar un sitio nos complican un poco la vida ya que cada explorador incluye las suyas propias y además no son iguales entre sí. Es 100% recomendable siempre usar un reset para así estandarizar el diseño al máximo de exploradores posibles.

En este caso estamos seleccionado todos los selectores a los cuales les vamos a aplicar un formato el cual se comvertirá en estandar para todos los exploradores que usemos para visializar la página web. 

	html, body, div, span, applet, object, iframe,
	h1, h2, h3, h4, h5, h6, p, blockquote, pre,
	a, abbr, acronym, address, big, cite, code,
	del, dfn, em, img, ins, kbd, q, s, samp,
	small, strike, strong, sub, sup, tt, var,
	b, u, i, center,
	dl, dt, dd, ol, ul, li,
	fieldset, form, label, legend,
	table, caption, tbody, tfoot, thead, tr, th, td,
	article, aside, canvas, details, embed, 
	figure, figcaption, footer, header, hgroup, 
	menu, nav, output, ruby, section, summary,
	time, mark, audio, video {
	margin: 0;
	padding: 0;
	border: 0;
	font-size: 100%;
	font: inherit;
	vertical-align: baseline;
	}

Para esta parte aplicaremos la propiedad display block a gran parte de las etiquetas semánticas y alguna que otra que queremos que se comporte por defecto en bloque.

	article, aside, details, figcaption, figure, 
	footer, header, hgroup, menu, nav, section {
	display: block;
	}

Bajo mi punto de vista todas las propiedades que se marcan en este reset desde "body" hasta "table" me gusta definirlas en el momento ya que se me puede dar el caso de que requiera otro estilo diferente y no pueda aplicar el mismo por culpa del reset inicial.

	body {
	line-height: 1;
	}

	ol, ul {
	list-style: none;
	}

	blockquote, q {
	quotes: none;

	}

	blockquote:before, blockquote:after,
	q:before, q:after {
	content: '';
	content: none;
	}	

	table {
	border-collapse: collapse;
	border-spacing: 0;
	}

Reset CSS diseñado por Eric A. and Kathryn S. Meyer.

Autoprefixers

¿Qué es un autoprefixer?

A mi entender un "Autoprefixers" es una pequeña aplicación web o algún tipo de Add-on/Plugin para tu IDE/herramienta de diseño el cual agrega de forma automática los prefijos de los motores de diseño de cada explorador los cuales aún no está estandarizado o implementado de forma permanente en su versión final.

  • -webkit
  • -moz
  • -o
  • -ms

Como ejemplo del mismo tenemos el website autoprefixer de gitnub
