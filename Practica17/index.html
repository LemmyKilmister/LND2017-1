Practica 17

Localización de recursos RSS

· Ventajas

La ventaja que más podemos destacar de un lector de RSS es su principal función. Dicha función no es otra que acercar de forma fácil y sencilla todo el contenido nuevo que se genera en nuestros sitios web favoritos sin la necesidad de estar visitándolo todos los días para ver si han añadido nuevo contenido al mismo.

Esta práctica vamos a usar el lector de contenido RSS realizado por Google, Podemos usar el que mejor nos venga, pero dado el gran crecimiento de Google como navegador más usado en la red usaremos su propio lector para una mayor integración con el navegador.

RSS en acción

Como ejemplo y buen uso de la tecnología del RSS tenemos el portal web "La moncloa", el cual además de agregar su propio contenido el mismo se auto completa de otros sitios web como "BOE" para indexar las nuevas incorporaciones del mismo a la web si necesidad de asistencia.
